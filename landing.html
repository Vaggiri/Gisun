Architecture & Design

A comprehensive overview of the GisunOS monorepo, tech stack, security boundaries, and licensing.

Stack Used

Unified Frontend & State

Built with React 18 and Vite for lightning-fast HMR. Style compilation is managed with Tailwind CSS and custom CSS modules. Dynamic spring physics and system animations are powered by GPU-accelerated Framer Motion, with lightweight state management handled via Zustand.

Core Bridge

Electron Shell & Express

In desktop mode, a secure local Express server acts as the OS bridge, running on loopback port 5000. It queries native files using Node's fs and launches default system viewers via child_process.exec when you click on local directories.

Deployment

Vercel & Web Fallback

The application automatically detects browser execution using process.versions.electron. When hosted on Vercel as a standard web app, native file features gracefully fall back to a mock Virtual File System (VFS), preventing cross-origin blockages.
